# Vaan 个人主页

这是 Vaan 的个人主页项目，采用现代化设计风格。

## 项目特点

- 🎨 现代化、美观的设计风格
- 📱 响应式布局，支持各种设备
- ⚡ 平滑滚动和动画效果
- 💬 访客留言功能（使用本地存储）
- 🎯 单页应用设计

## 项目结构

```
vaanai/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── script.js      # JavaScript 脚本
├── images/
│   ├── avatar.jpg     # 头像图片
│   └── wechat.jpg     # 微信二维码
└── README.md          # 项目说明
```

## 功能模块

1. **关于我** - 个人介绍
2. **我的技能** - 技能展示
3. **我的作品** - 作品集展示
4. **愿景使命** - 个人愿景
5. **联系方式** - 邮箱和微信
6. **访客留言** - 留言板功能

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器运行项目

## 技术栈

- HTML5
- CSS3（渐变、动画、Grid布局）
- 原生 JavaScript
- LocalStorage（留言存储）

## 浏览器支持

支持所有现代浏览器：
- Chrome
- Firefox
- Safari
- Edge
